How to Play Numberfive

Numberfive is a numerical puzzle game where you place missing numbers into a grid.


Each row and column has a sum displayed, and you need to fill in the missing numbers so that each row and column sums to the displayed total.

How to Play:

  1. Click on any empty space in the grid
  2. Click a number button to place that digit in the selected space
  3. Continue until you've filled all empty spaces
  4. If all row and column sums match, you win!
